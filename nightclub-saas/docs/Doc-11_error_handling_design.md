---

## 2. エラーレスポンス標準形式

```json
{
  "success": false,
  "error": {
    "code": "AUTH_001",
    "message": "ログインに失敗しました",
    "detail": "Invalid password",
    "correlationId": "uuid"
  }
}
```

| フィールド | 用途 |
|---|---|
| code | 業務エラーコード |
| message | ユーザー向け日本語 |
| detail | 内部詳細（開発用） |
| correlationId | ログ追跡用 |

---

## 3. HTTPステータス方針

| コード | 用途 |
|---|---|
| 401 | 認証失敗 |
| 403 | 権限不足 |
| 404 | データ未存在 |
| 409 | 状態競合（締め済み等） |
| 422 | 入力不正 |
| 429 | レート制限 |
| 500 | 想定外例外 |

---

## 4. 業務エラーコード体系

### AUTH系

| コード | 説明 |
|---|---|
| AUTH_001 | 認証失敗 |
| AUTH_002 | トークン期限切れ |
| AUTH_003 | アカウント無効 |

### TENANT系

| コード | 説明 |
|---|---|
| TENANT_001 | tenant不一致 |
| TENANT_002 | tenant未設定 |

### RBAC系

| コード | 説明 |
|---|---|
| ACCESS_001 | 権限不足 |
| ACCESS_002 | 自分以外参照不可 |

### 締め系

| コード | 説明 |
|---|---|
| CLOSE_001 | 日次締め済み |
| CLOSE_002 | 月次確定済み |
| CLOSE_003 | 未入力データあり締め不可 |

### 勤怠系

| コード | 説明 |
|---|---|
| PUNCH_001 | 二重打刻 |
| PUNCH_002 | 未退勤 |
| PUNCH_003 | 未あがり |

### 売上系

| コード | 説明 |
|---|---|
| SALES_001 | 伝票未存在 |
| SALES_002 | 締め済み修正禁止 |

### 給与系

| コード | 説明 |
|---|---|
| PAYROLL_001 | 期間重複 |
| PAYROLL_002 | 不正バック率 |
| PAYROLL_003 | 計算対象未確定 |

### 修正申請系

| コード | 説明 |
|---|---|
| CHANGE_001 | 承認待ち重複 |
| CHANGE_002 | 却下済み |

### 入力系

| コード | 説明 |
|---|---|
| VALID_001 | 必須項目未入力 |
| VALID_002 | 不正形式 |
| VALID_003 | 数値範囲外 |

### システム系

| コード | 説明 |
|---|---|
| SYS_001 | 想定外エラー |
| SYS_002 | DB接続失敗 |

---

## 5. フロント表示文言設計

| コード | ユーザー向け表示 |
|---|---|
| CLOSE_001 | 締め済みデータは編集できません。修正申請を行ってください。 |
| ACCESS_001 | この操作を行う権限がありません。 |
| PUNCH_001 | 既に出勤打刻されています。 |
| VALID_001 | 入力必須項目が不足しています。 |

---

## 6. 例外分類

| 分類 | 説明 | ログレベル | 通知 |
|---|---|---|---|
| 業務例外 | 想定内エラー | INFO | - |
| セキュリティ例外 | 権限突破試行等 | WARN | ○ |
| システム例外 | DB接続障害等 | ERROR | 即時アラート |

---

## 7. ログ出力仕様

全エラーで以下をログ出力:
- timestamp、tenantId、accountId、endpoint、errorCode、correlationId

> 重大例外のみ通知送信

---

## 8. バリデーション方針

- API層で必須/型/範囲チェック
- 締め状態チェック
- tenant一致チェック
- バリデーション失敗は **422**

## 9. テスト項目

| テスト | 検証内容 |
|---|---|
| 締め済み編集禁止 | 409 + CLOSE_001 |
| Cast給与APIアクセス不可 | 403 + ACCESS_001 |
| tenant混線 | 403 + TENANT_001 |
| 二重打刻 | 409 + PUNCH_001 |
| 不正バック率 | 422 + PAYROLL_002 |
| 想定外例外 | 500 + SYS_001 |
